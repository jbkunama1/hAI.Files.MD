hAI.Files.MD

Self-hosted Docker stack for files.md – with PWA frontend and sync server.

Language / Sprache: 🇩🇪 Deutsch · 🇬🇧 English

Quick start

Clone this repository and the upstream files.md repo on your server, then deploy the stack with Docker Compose or Portainer.

git clone https://github.com/jbkunama1/hAI.Files.MD.git
cd hAI.Files.MD

# adjust docker-compose.yml (HOST, SALT)

docker compose build
docker compose up -d

Open the PWA at http://YOUR-SERVER:8080 and point it to your sync server at http://YOUR-SERVER:3333 via localStorage.setItem('ApiHost', 'http://YOUR-SERVER:3333') in the browser console.
